Métricas de Calidad en el Desarrollo de Software



Las métricas de calidad son medidas cuantitativas que se utilizan para evaluar la calidad de un producto o servicio. Se basan en características clave que definen el cumplimiento de los requisitos y las expectativas del cliente la elección de las métricas de calidad adecuadas depende de las necesidades específicas que requieran para el producto o servicio.

Tipos de Métricas de Calidad


  • Métricas de tamaño

    Miden el tamaño del software, como el número de líneas de código o el número de funciones.

  • Métricas de complejidad

    Miden la complejidad del software, como el número de caminos de control o el número de dependencias entre módulos.

  • Métricas de defectos

    Miden el número de defectos encontrados en el software, como errores de código o fallos en la interfaz de usuario.

  • Métricas de rendimiento

    Miden el rendimiento del software, como el tiempo de respuesta o la utilización de recursos.

  • Métricas de pruebas

    Miden la cobertura de las pruebas, la tasa de éxito de las pruebas y el tiempo de ejecución de las pruebas.

  • Métricas de mantenimiento

    Miden el esfuerzo y el tiempo necesarios para mantener el software, como la tasa de corrección de errores o el número de cambios realizados en el código.
